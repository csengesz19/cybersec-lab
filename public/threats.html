Common Web Attacks

Overview of typical web-focused attacks you might encounter when studying web security.

Types of Attacks

Below are common categories — each has different vectors and mitigations.

  • Cross-Site Scripting (XSS) — injecting scripts into pages viewed by others.
  • SQL Injection (SQLi)— injecting queries to manipulate databases.
  • Cross-Site Request Forgery (CSRF) — forging actions on behalf of logged-in users.
  1. Discover input point
  2. Craft payload
  3. Deliver payload
  4. Escalate and persist

Example: Simple XSS

The following illustrates how an unsanitized input could be abused:

<input name="q" value="<script>alert('XSS')</script>">
